# Romualdi Lab

Welcome to the **Romualdi Lab** website repository.

The Romualdi Lab, based at the University of Padova, focuses on **computational biology**, **bioinformatics**, and **genomics**, with a particular interest in the analysis and interpretation of complex biological data.

This website presents an overview of our **research projects**, **people**, **software tools**, and **publications**, as well as opportunities to **join the group**.
